About

App description:

This simple JavaScript App makes use of the Teleport Public API to fetch data in relation to the quality of life in different cities worldwide.

The main page shows a list of main cities worldwide and then the User can click on one city to get access to a summary of information provided by the Teleport Public API.

Once the User clicked on a city, he can also get acces to a link to further details displayed directly on Teleport Website.

This is a great way to get to discover new cities to live and work!

App dependencies:

  • JavaScript
  • Bootstrap
  • jQuery

API used:

  • Teleport Public API (AIC API)
